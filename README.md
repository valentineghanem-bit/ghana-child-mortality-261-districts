# Subnational Spatial Distribution and Ensemble ML Predictors of Neonatal and Under-Five Mortality Across Ghana's 261 Health Districts

**Author:** Valentine Golden Ghanem | Ghana COCOBOD Cocoa Clinic, Accra | ORCID: 0009-0002-8332-0220

## Overview
An ecological study integrating spatial epidemiology (Global Moran's I, LISA, bivariate Moran's I) and ensemble machine learning (Random Forest, Gradient Boosting, stacked ensemble + Gini feature importance) to characterise the subnational distribution and determinants of neonatal and under-five mortality across all 261 Ghanaian health districts using 2022 DHS and 2021 Census data.

## Key Findings
- U5MR ranges from 20.0 (Greater Accra) to 72.0 (Oti) per 1,000 live births — a 3.6× north–south gradient
- NMR ranges from 8.0 to 23.0 per 1,000 live births (median: 17.0 [IQR: 12.5–18.0])
- 72 high-risk districts (27.6%) classified by top-quartile U5MR (≥48 per 1,000 LB)
- Significant spatial clustering confirmed by Global Moran's I = 0.7783 (z=19.322, p<0.001, KNN-4)
- Top predictors (Gini, RF): early breastfeeding initiation (0.224), diarrhoea prevalence (0.182), full vaccination coverage (0.136)
- Stacked ensemble CV AUC: 1.00 (note: ecological artifact — DHS predictors assigned uniformly across 16 regions; reflects regional-scale separation, not district-level individual risk)

## Methods
- **Spatial autocorrelation:** Global Moran's I (KNN-4, 999 permutations) + LISA (KNN-4 weights, p<0.05) + Bivariate Moran's I
- **ML:** Random Forest (RF) + Gradient Boosting (GB) → Stacked Ensemble (logistic meta-learner); 10-fold stratified CV; scikit-learn v1.5.2
- **Interpretability:** Gini impurity (RF base learner); top features: early breastfeeding initiation (0.224), diarrhoea prevalence (0.182), full vaccination coverage (0.136)
- **Reporting:** STROBE (observational, ecological design)
